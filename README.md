# sites-500-pages

本仓库用于归档与发布多个独立 HTML 页面。这些页面彼此独立，不针对任何特定域名或网站。

## 项目简介

本项目旨在提供一个结构化的空间，用于存放和展示一批静态 HTML 页面。页面内容涵盖多种类型，作为通用归档与演示用途。仓库本身不涉及任何商业推广或定向访问引导。

## 目录结构

```
sites-500-pages/
├── index.html          # 页面索引（如有）
├── pages/              # 存放主要 HTML 页面文件
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md
```

- **pages/**：存放所有独立 HTML 页面文件，按编号命名以便管理。
- **assets/**：存放页面共享的样式、脚本与图片资源。
- **index.html**（可选）：若存在，可作为页面列表或导航入口。

## 页面归档说明

每个 HTML 页面均为独立完整的静态文件，不依赖外部服务或数据库。页面之间无链接跳转关系，各自展示不同内容。归档内容以通用性为主，不包含具体商业信息或定向推广元素。

## 使用方式

您可以通过以下方式浏览或引用这些页面：

1. **直接浏览**：将仓库克隆到本地，用浏览器打开 `pages/` 目录下的任意 HTML 文件。
2. **在线预览**：通过 GitHub Pages 或其他静态托管服务启用后，可直接访问对应页面 URL。
3. **引用嵌入**：可将单个页面嵌入到其他项目或文档中（需遵守开源许可）。
